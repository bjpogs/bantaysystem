WEB-BASED INFORMATION AND MANAGEMENT SYSTEM OF BARANGAY BANTAY, BOAC, MARINDUQUE

----------------------------------------------------------------------------------------------------------------------------------------

HOW TO USE:
- get 1 glass of water
- Install XAMPP control panel
- copy project folder to "C:\xampp\htdocs"
- open Chrome or any browser
- go to "localhost/phpmyadmin"
- create database named "brgysystem"
- open brgystem database
- select import and go to "C:\xampp\htdocs\chumchum\assets\files" and select brgysystem.sql
- proceed to "localhost/projectname | example : "localhost/brgysystem"
- log in and drink glass of water.
